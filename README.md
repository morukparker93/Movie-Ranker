# Movie-Ranker

A web application to discover, rank, and organize movies through side-by-side comparisons.

This project is a fork of [periculum/Movie-Ranker](https://github.com/periculum/Movie-Ranker). 
The main goal of this fork was to achieve much better movie recognition. It now integrates The Movie Database (TMDB) API to correctly identify movies and automatically fetch rich metadata, including accurate release years, directors, and high-quality movie posters.

## Features
- **Movie Ranking:** Compare two movies and choose your favorite to build a personal ranking list.
- **TMDB Integration:** Reliable movie recognition with automated metadata and poster fetching.
- **Responsive UI:** Clean, simple, and mobile-friendly interface for an optimal ranking experience.
